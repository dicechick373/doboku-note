# 費用負担者

## 道路の管理に関する費用負担の原則

道路の管理に関する費用は、道路法等における特別の定めを除き、原則として当該道路の道路管理者の負担である（[道路法第49条](https://elaws.e-gov.go.jp/document?lawid=327AC1000000180#Mp-At_49)）。

| 道路区分 | 道路管理者 |
| ---- | ---- |
| 国道（指定区間内） | 国 |
| 国道（指定区間外） | 都道府県 |
| 都道府県道 | 都道府県 |
| 市町村道 | 市町村 |


## 道路の管理に要する費用の意義

道路の管理に要する費用とは、およそ道路の管理権の作用として行われる一切の行為に要する費用をいう。

## 道路の管理に関する費用負担の原則に対する特例

道路の管理に関する費用負担の原則に対する特例は、[道路法](https://elaws.e-gov.go.jp/document?lawid=327AC1000000180)、[公共土木施設災害復旧事業費国庫負担法](https://elaws.e-gov.go.jp/document?lawid=326AC0000000097)、その他の法律において規定されている。

## 高速自動車国道の管理に要する費用

高速自動車国道の管理に要する費用は、国が4分の3を、都道府県が4分の1を負担する（[高速自動車国道法第20条第1項](https://elaws.e-gov.go.jp/document?lawid=332AC0000000079#Mp-At_20)）が、現在高速自動車国道は、有料道路として機構及び会社において管理されているので、当該費用は当該機構及び会社の負担である（[道路整備特別措置法第37条](https://elaws.e-gov.go.jp/document?lawid=331AC0000000007#Mp-At_37)）。

## Q and A

### 他の法律における特例

Q 道路法による道路の管理に関する費用負担の原則に対する特例を規定するものとして、どのような法律があるか。

A [道路整備事業に係る国の財政上の特別措置に関する法律](https://elaws.e-gov.go.jp/document?lawid=333AC0000000034_20230906_505AC0000000043)（昭和33年法律34号）、[道路整備特別措置法](https://elaws.e-gov.go.jp/document?lawid=331AC0000000007)（昭和31年法律7号）、[道路の修繕に関する法律](https://elaws.e-gov.go.jp/document?lawid=323AC1000000282)（昭和23年法律282号）、[公共土木施設災害復旧事業費国庫負担法](https://elaws.e-gov.go.jp/document?lawid=326AC0000000097)（昭和26年法律97号）、[積雪寒冷特別地域における道路交通の確保に関する特別措置法](https://elaws.e-gov.go.jp/document?lawid=331AC1000000072_20150801_000000000000000)（昭和31年法律72号）、[共同溝の整備等に関する特別措置法](https://elaws.e-gov.go.jp/document?lawid=338AC0000000081_20220401_502AC0000000049)（昭和38年法律81号）、[交通安全施設等整備事業の推進に関する法律](https://elaws.e-gov.go.jp/document?lawid=341AC0000000045)（昭和41年法律45号）、[離島振興法](https://elaws.e-gov.go.jp/document?lawid=328AC1000000072)（昭和28年法律72号）、[奄美群島振興開発特別措置法](https://elaws.e-gov.go.jp/document?lawid=329AC1000000189_20220617_504AC0000000068)（昭和29年法律189号）、[沖縄振興特別措置法](https://elaws.e-gov.go.jp/document?lawid=414AC0000000014)（平成14年法律14号）等がある。

### 管理瑕疵による損害賠償費用

Q 道路の設置又は管理の瑕疵による損害賠償費用は、道路の管理に関する費用に含まれると解して差し支えないか。

A 差し支えない（「一般国道の設置又は管理の瑕疵により生じた損害に係る損害賠償費用の負担について」昭和44年9月25日内閣法制局一発5号道路局長あて内閣法制局第一部長回答参照）。
