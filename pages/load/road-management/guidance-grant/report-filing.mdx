# 報告の提出

## 報告を要する場合

道路管理者は、都道府県である場合にあっては国土交通大臣に、市町村である場合にあっては都道府県知事に、それぞれの要領で次に定める事項を、報告しなければならない（法76、施行規則10I）。

1. 道路整備計画については、社会経済情勢の変化等に伴い道路整備計画を作成し、又は変更したつど速やかに
2. 道路に関する工事の施行実績については、工事を施行した後速やかに
3. 道路と鉄道とが交差する場合における交差方式等についての協議の内容については、協議が成立したつど速やかに
4. 占用料条例、連結料条例又は受益者負担金条例については、条例を制定したつど速やかに

なお、「国土交通大臣の認可」の項で述べたように、認可の対象とならない軽微な事項については、報告の義務があること（施行規則9Ⅱ）に注意。

また、道路管理者は、前記1の道路整備計画についての報告を行うときは、一定の様式により、都道府県にあっては縮尺5万分の1程度の、市町村にあっては都道府県が市町村ごとに定める縮尺（5万分の1以上のものに限る。）の図面に少なくとも次に掲げる事項を記載したものを添付して行うものとする（施行規則10Ⅱ）。

1. 市町村、大字及び字の名称並びに境界線
2. 車道の幅員
3. 主要なトンネル、橋及び渡船施設並びにこれらの名称
4. 道路と効用を兼ねる主要な他の工作物
5. 交差し、若しくは接続する道路又は重複する道路のうち主要なもの並びにこれらの種類及び路線名
6. 交差する鉄道又は新設軌道及びこれらの名称
7. 作成の年月日
